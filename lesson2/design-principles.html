Design Principles

Lesson 2

Visual Hierarchy

O'Reilly Library

The O'Reilly Library shows good visual hierarchy because it easily guides the viewers eyes. The viewer would first see the image, then the title, and then it would continue to follow the text based on text size and color.

Fitt's Law

Hulu

Hulu's website is a good example of Fitt's Law because the bigger elements on the page appears to be easier to access then some of the smaller elements.

Alignment

Pinterest

Pinterest is a good representaiton of alignment because the elements on the page seem to be set up on a grid and both sides have the same amount of white space keeping everything lined up nicely.
