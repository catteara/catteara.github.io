Typography

For my website, I chose the fonts DM Serif Display to use for my webpage name and then I have decided to use Gothic A1 for everything else. Both of these fonts are from fonts.google.com. As of now, the only change I would consider making to this decision is if I decide the main portion of my page needs more contrast, in which case I would change my headings to DM Serif Display as well.
